# LogView

#### 项目介绍
ws: tail -f 查看服务器日志功能 

#### 使用方式
上个图 spring boot 项目直接 java -jar LogView.jar 启动即可,没用数据库!右侧为服务器当前的路径的文件，文件夹可以点击！想看哪个日志，
结合上面的路径放入待查看Input查看。http://127.0.0.1:7003/logView/ 回车就可访问
